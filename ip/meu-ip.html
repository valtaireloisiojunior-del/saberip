O que é um endereço IP?

O endereço IP (Internet Protocol) é um identificador único atribuído à sua conexão quando você acessa a internet. Ele permite que dispositivos se comuniquem entre si em redes locais e na internet.

Qual é meu IP público?

O IP público é o endereço visível na internet e fornecido pelo seu provedor. É ele que sites e serviços online utilizam para identificar sua conexão. Esta ferramenta mostra seu IP público em tempo real.

Meu IP muda?

Na maioria dos casos, sim. Provedores de internet geralmente utilizam IP dinâmico, o que significa que seu endereço IP pode mudar sempre que o modem é reiniciado ou após determinado período.

IP público x IP privado

O IP público é usado para comunicação na internet. Já o IP privado é utilizado apenas dentro da sua rede local (como 192.168.0.1).
